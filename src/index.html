Zaczynamy od stworzenia pliku package.json w katalogu głównym naszego projektu przy pomocy komendy: npm init -y
Potem instalujemy niezbędne paczki:
npm i webpack@4 -D
npm i webpack-cli@3 -D
npm i webpack-dev-server@3 -D
npm i html-webpack-plugin@3 -D
npm i @babel/core@7 -D
npm i @babel/preset-env@7 -D
npm i babel-loader@8 -D
Połączenie z data.json, w konsoli cmd folder src komenda: json-server ./data.json --watch
Odpalenie strony, w konsoli cmd folder api-crud komenda: npm start
